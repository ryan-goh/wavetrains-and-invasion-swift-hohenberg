Directory contains files for/from using the numerical continuation software auto for creating the figure in the book chapter about existence of wavetrains. 
For this purpose open the file "plot.auto" in a text editor and copy paste the commands into the auto python environment. 
Then open "plotting_auto.m" and follow the commands.

The files s.xxx and b.xxx contain the results of continuation with auto based on the files provided, with suitable steps and choices for the numerical constants in the c.xxx file. 
We omit the details.
